# RS NOTES 产品需求文档（PRD）

## 1. 产品概述

RS NOTES 是一个面向科研工作者与技术开发者的个人知识管理网站，以极简黑白美学为核心语言，承载科研展示、笔记沉淀与经验分享三大内容形态。
- 目标用户：科研人员、工程师、技术写作者，需要一个长期沉淀与对外展示的统一入口
- 价值定位：用最克制的视觉语言突出内容本身，让科研成果、技术笔记、生活分享在同一个黑白系统内共存

## 2. 核心特性

### 2.2 功能模块

1. **首页（Home）**：Three.js 粒子全屏背景 + Hero 欢迎区（头像 / Logo / 欢迎语）+ 第二屏 Research / Notes / Share 三模块入口
2. **科研页（Research）**：项目墙展示与详情页
3. **笔记页（Notes）**：Markdown 渲染、TOC、标签筛选、搜索、阅读统计
4. **分享页（Share）**：时间轴 + 图片瀑布流 + 预览 + 搜索
5. **全站搜索系统**：极简毛玻璃搜索框，点击展开
6. **配置系统**：站点信息集中配置

## 3. 核心流程

**主流程一：访客浏览首页并进入模块**
访客打开站点 → 粒子背景渲染 → Hero 入场动画 → 滚动至第二屏 → 点击 Research / Notes / Share 卡片 → 跳转对应页

**主流程二：阅读一篇笔记**
访客进入笔记页 → 浏览标签云与列表 → 输入关键词搜索或点击标签筛选 → 点击文章 → 进入详情页 → LocalStorage 记录阅读次数 → 显示阅读时长估算与 TOC

**主流程三：使用全站搜索**
任意页面点击搜索图标 → 搜索框平滑展开 → 输入关键词 → 实时筛选科研/笔记/分享三类结果 → 点击跳转 → Esc 关闭

## 4. 用户界面设计

### 4.1 设计风格

- **主色调**：纯黑 `#0a0a0a` 背景 + 纯白 `#fafafa` 文字 + 灰阶层次（`#1a1a1a` / `#666` / `#999` / `#e5e5e5`），强调色仅用纯白
- **字体策略**：标题使用具备学术与几何感的展示字体（如 Fraunces 衬线变体或 IBM Plex Mono），正文使用 IBM Plex Sans / 思源黑体，代码与数据使用 JetBrains Mono
- **按钮风格**：极简线框按钮，1px 白边框，悬停反色填充，无圆角或极小圆角（2px）
- **布局风格**：编辑性杂志排版，大量留白，左右不对称栅格，内容居中容器最大宽度 1200px
- **图标风格**：线性 1px 单色图标，无填充，与极简调性一致
- **纹理细节**：叠加极轻的颗粒噪点（SVG noise），增加质感与深度，避免纯色平面

### 4.2 页面设计概述

| 页面名称 | 模块名称 | UI 元素 |
|----------|----------|----------|
| 首页 | 粒子背景 | 黑底白点粒子网络，鼠标轻微交互， opacity 0.6 |
| 首页 | Hero 区 | 头像（圆形描边）+ Logo + 欢迎语大标题，GSAP 淡入上浮 |
| 首页 | 第二屏 | 三列卡片网格，悬停时卡片描边变亮并显示预览缩略 |
| 科研页 | 项目墙 | 3 列响应式网格卡片，封面顶部 + 标签胶囊 + 状态点 |
| 笔记页 | 列表区 | 左 1/4 标签云 + 右 3/4 文章列表，每条含标题/摘要/阅读统计 |
| 分享页 | 时间轴 | 左侧垂直时间线 + 右侧内容块，图片瀑布流采用 CSS columns |
| 全站 | 搜索框 | 顶栏毛玻璃，搜索框展开宽度从 0 → 320px，细边框流光动画 |

### 4.3 响应式

- 桌面优先（≥ 1024px）完整三列布局与粒子效果
- 平板（768–1024px）两列网格，第二屏改为纵向堆叠
- 移动端（< 768px）单列堆叠，导航折叠为汉堡菜单，粒子数量降级以保证性能

### 4.4 3D 场景说明（Three.js 粒子背景）

- 环境：纯黑底，白色粒子点，粒子之间近距离连线形成网络
- 相机：正对屏幕的透视相机，轻微随鼠标视差偏移
- 运动：粒子缓慢漂移，超出边界后从对侧回归
- 性能预算：桌面约 120 粒子 + 连线，移动端约 40 粒子，关闭连线
- 后处理：不使用重量级后处理，仅靠透明度与混合模式营造氛围

## 5. 阅读统计系统

- **阅读次数**：使用 LocalStorage 键 `article_views_{id}`，每次进入详情页自增
- **阅读时长估算**：按 200 字/分钟，根据正文总字数计算分钟数，向上取整
- **显示位置**：笔记列表卡片与笔记详情页顶部，同时展示阅读次数与预计阅读时间

## 6. 配置系统

- 路径：`/config/site.config.js`
- 可配置项：Logo、Avatar、作者信息、社交链接、导航菜单、站点标题与副标题
- 全站通过 `window.SITE_CONFIG` 读取，统一渲染
